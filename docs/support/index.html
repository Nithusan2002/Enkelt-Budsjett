Vanlige spørsmål

Hvordan sletter jeg alle data?

Gå til Innstillinger i appen og velg alternativet for å slette alle data.

Kan jeg eksportere data?

Ja. Du finner eksport og import under Innstillinger. Eksport lages som en JSON-fil.

Synkroniseres data mellom enheter?

Hvis iCloud er aktiv på enheten, kan appen synkronisere via CloudKit. Uten iCloud lagres data lokalt.
