Spor økonomi · Sist oppdatert: 13. mars 2026

Kort fortalt

Spor økonomi er laget for å gi deg enkel oversikt over økonomien din uten bankintegrasjon i MVP. Data lagres lokalt på enheten som standard, og appen bruker ikke tredjepartssporing eller annonsering i denne versjonen.

Hvilke data behandles

  • Appen behandler dataene du selv legger inn, som inntekter, utgifter, faste poster, mål, investeringsoversikt og innstillinger.
  • Hvis du velger å opprette konto eller logge inn, behandles opplysninger som e-postadresse, bruker-ID og eventuelt visningsnavn for autentisering og kontohåndtering.
  • Dataene brukes kun for appfunksjonalitet og for å gi deg tilgang til funksjonene du velger å bruke.

Lagring

  • Data lagres lokalt på enheten med SwiftData som standard.
  • Hvis iCloud-synk er aktivert på enheten din, kan appdata synkroniseres via Apple sine tjenester knyttet til Apple-kontoen din.
  • Hvis du bruker konto, lagres innloggingsøkt sikkert på enheten, og autentiseringsdata behandles av vår autentiseringsleverandør for innlogging og kontohåndtering.

Deling med tredjeparter

  • Spor økonomi bruker ikke tredjepartssporing i denne versjonen.
  • Spor økonomi bruker ikke annonserings-SDK-er i denne versjonen.
  • Hvis du velger å bruke konto, behandles innloggingsdata av autentiseringsleverandøren vår.

Eksport og sletting

  • Du kan eksportere data som JSON-fil fra Innstillinger.
  • Du kan slette alle lokale data fra Innstillinger.
  • Hvis du bruker konto, kan du også be om sletting av konto fra appen.
