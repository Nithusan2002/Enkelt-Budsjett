Spor økonomi · Sist oppdatert: 6. mars 2026

Kort fortalt

Spor økonomi er laget for å gi deg budsjett- og investeringsoversikt uten bankintegrasjon i MVP.

Hvilke data behandles

  • Budsjettdata, transaksjoner, mål og innstillinger du legger inn i appen.
  • Hvis du velger å opprette konto eller logge inn med e-post, behandles e-postadresse, bruker-ID og eventuelt visningsnavn for autentisering og kontohåndtering.
  • Disse dataene brukes kun for appfunksjonalitet.

Lagring

  • Data lagres lokalt på enheten med SwiftData.
  • Hvis iCloud-synk er aktiv, kan data synkroniseres via CloudKit i din Apple-konto.
  • Ved bruk av konto lagres innloggingsøkt sikkert på enheten, og autentiseringsdata behandles av vår autentiseringsleverandør for å logge deg inn.

Deling med tredjeparter

  • Ingen tredjepartssporing i MVP.
  • Ingen annonserings-SDK-er i MVP.
  • Vi bruker autentiseringstjeneste for konto når du velger å logge inn med e-post.

Eksport og sletting

  • Du kan eksportere data som JSON-fil fra Innstillinger.
  • Du kan slette alle lokale data fra Innstillinger.
